---
id: feature-matrix
title: OSS vs Enterprise
---

| Capability | OSS | Enterprise |
|---|---|---|
| Multi-vendor gateway | Yes | Yes |
| Team analytics by user/project/department | Yes | Yes |
| Basic audit trail | Yes | Yes |
| OIDC/SSO enterprise connectors | No | Yes |
| Approval workflow and policy engine | No | Yes |
| Compliance report templates and governance exports | No | Yes |
| Multi-cluster control operations | No | Yes |
| Enterprise SLA and support tooling | No | Yes |

ZSGate keeps OSS fully usable for team-level adoption, while Enterprise focuses on governance depth.
