# Websites Monorepo

Unified monorepo for:
- `xfa-website` (`xfa.cn`)
- `zhsjf-website` (`zhsjf.cn`)
- `aihunt-website` (`aihunt.com`)
- `zsgate-website`

## Stack
- TanStack Router + TanStack Query + React + TypeScript
- Vite
- pnpm + Turborepo
- Vitest + Playwright
- Docker (self-hosted)

## Quickstart
```bash
pnpm install
pnpm dev:xfa
```
